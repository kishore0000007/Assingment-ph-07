 # 🌿 KeenKeeper

A modern web app to help you stay connected with your friends by tracking interactions like calls, texts, and video chats.

---

## 🚀 Features

- 📇 Manage and view your friends
- ⏱ Track days since last contact
- 📊 Visual analytics with charts
- 🕒 Timeline of interactions
- 🔔 Toast notifications for actions
- 📱 Fully responsive design
- ❌ Custom 404 page handling

---

## 🛠 Technologies Used

| Technology | Purpose |
|-----------|--------|
| **Next.js (React)** | Build the UI |
| **App Router (Next.js)** | Handle routing |
| **Tailwind CSS** | Styling and responsiveness |
| **Recharts** | Data visualization (charts) |
| **React Toastify** | Notifications |
| **Lucide React** | Icons |
